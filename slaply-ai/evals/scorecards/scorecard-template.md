## Rate 1-5

1. Scope control: Did the report stay inside Text Errors / Hierarchy / Readability?
2. Text Errors accuracy: Did it catch real visible text issues without inventing errors?
3. Hierarchy accuracy: Did it only flag concrete pre-flight information risks?
4. Readability accuracy: Did it avoid small-text critique and focus on real unreadability / watermark / QR / low-res risks?
5. Annotation accuracy: Are markers on the exact visible issue?
6. Severity accuracy: Are High / Medium / Low labels appropriate?
7. Language quality: Is the report fully in the selected language except exact artwork quotes?
8. Boundary safety: No FDA / Thai FDA / legal / print-ready / sales guarantee claims?
9. Customer usefulness: Would the customer know what to check or fix?
10. False positives: Are unnecessary issues low?

Pass / Fail:
