# Full Case Bank

The Full Case Bank represents 500+ private packaging artwork projects used for sampling and edge-case mining.

Do not treat this folder or `../templates/case_bank.csv` as the active eval set.

Use it to:

- keep sanitized metadata and placeholder artwork references
- tag interesting failure patterns
- identify candidates for the Golden Set or Regression Set
- preserve coverage across category, language, artwork format, and known issue types

Do not store real artwork files in this repo. Store real artwork only in private/local storage and reference it with `private://`, `local-only://`, `icloud-private://`, or `r2-private://` placeholders.
