# Golden Set

The Golden Set is the small, manually reviewed set used after every prompt change.

Target size:

- 30-50 active cases

Use it to decide whether a prompt change improves Slaply's practical report quality while staying inside:

- `Text Errors`
- `Hierarchy`
- `Readability`

Each active Golden case should have:

- complete metadata
- a private artwork reference
- known expected behavior
- a completed scorecard for each prompt version reviewed
- clear `must_not_flag` values

Do not add real artwork files here.
