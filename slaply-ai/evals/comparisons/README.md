# Prompt Output Comparisons

Use this folder for human-readable comparisons between prompt versions.

Suggested layout:

```text
comparisons/
  slaply-preflight-v0.1__vs__slaply-preflight-v0.2/
    SKINCARE_001.md
```

Each comparison should summarize:

- issues added
- issues removed
- severity changes
- annotation changes
- report language changes
- practical recommendation changes
- any new `must_not_flag` violation

Keep comparisons sanitized. Do not paste real artwork, customer names, customer emails, private URLs, or sensitive customer text.
