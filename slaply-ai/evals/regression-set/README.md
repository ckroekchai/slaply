# Regression Set

The Regression Set is the broader check before deploy or before accepting a prompt version.

Target size:

- 100-150 active cases

Use it to catch repeated regressions across common packaging categories, report languages, and known failure patterns.

Blocking failures include:

- broad design critique
- premium or beauty/taste judgment
- compliance or FDA / Thai FDA approval language
- print-ready or manufacturing approval language
- sales guarantee language
- severe annotation mismatch
- report language mismatch

Do not add real artwork files here.
